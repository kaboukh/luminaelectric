## ✨ AMÉLIORATIONS VISUELLES AJOUTÉES

### Effets hover
```css
.logo:hover {
  transform: scale(1.05);      /* Agrandissement 5% */
  opacity: 0.9;                /* Légère transparence */
}

.logo:hover img {
  transform: translateY(-2px); /* Léger lift */
}
```

**Résultat** : Le logo "réagit" au survol, effet premium

### Footer hover
```css
.footer-logo:hover {
  transform: scale(1.05);      /* Agrandissement */
}

.footer-logo:hover img {
  filter: brightness(1.1);     /* Éclaircissement */
}
```

**Résultat** : Le logo s'illumine au survol
