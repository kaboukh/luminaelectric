### Étape 1 : Créer compte Netlify
1. Aller sur https://netlify.com
2. S'inscrire (gratuit)
3. Vérifier email

### Étape 2 : Déployer le site
1. Cliquer "Add new site" → "Deploy manually"
2. Glisser-déposer le dossier `luminaelectric.be/`
3. Attendre 2-3 minutes
4. Récupérer URL temporaire (ex : `luminaelectric-xyz.netlify.app`)

### Étape 3 : Tester
1. Tester formulaire (confirmation email nécessaire 1ère fois)
2. Tester liens téléphone/WhatsApp
3. Vérifier responsive mobile
4. Tester navigation

### Étape 4 : Domaine personnalisé
1. Acheter `luminaelectric.be` (~15 €/an)
2. Configurer DNS :
   ```
   Type: A
   Name: @
   Value: 75.2.60.5
   
   Type: CNAME
   Name: www
   Value: luminaelectric-xyz.netlify.app
   ```
3. Attendre propagation DNS (2-48h)
4. Activer HTTPS automatique (Netlify)
