### Option 1 : Chrome DevTools (PC)
1. Ouvrir `nl/over-ons.html` ou `nl/contact.html` dans Chrome
2. Appuyer sur **F12** (outils développeur)
3. Cliquer sur l'icône **📱 Toggle device toolbar** (Ctrl+Shift+M)
4. Sélectionner **"iPhone SE"** (375 × 667) ou **"iPhone 12 Pro"** (390 × 844)
5. Cliquer sur le menu hamburger **☰**
6. Vérifier que les liens sont **bien lisibles en noir** :
   - Home
   - Diensten
   - Realisaties
   - FAQ
   - **Over ons** (actif = bleu)
   - **Contact**

### Option 2 : Smartphone réel
1. Ouvrir le site sur votre smartphone
2. Appuyer sur le menu hamburger **☰**
3. Vérifier la lisibilité des liens
