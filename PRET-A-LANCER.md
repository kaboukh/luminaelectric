- 👉 https://combell.com

**Option D - OVH** (Européen)
- Payant (~40€/an)
- 👉 https://ovh.com

### 2️⃣ Uploader les fichiers

**Netlify (le plus simple)** :
1. Créer un compte sur netlify.com
2. Glisser-déposer TOUT le dossier `luminaelectric.be/`
3. Site en ligne en 30 secondes !
4. URL temporaire : `votre-site.netlify.app`

**Autres hébergeurs** :
1. Se connecter via FTP (FileZilla)
2. Uploader tous les fichiers
3. Vérifier que `index.html` est à la racine

### 3️⃣ Configurer le domaine

**luminaelectric.be** :
1. Acheter le domaine (si pas encore fait)
   - Chez Combell, OVH, GoDaddy...
   - ~15€/an

2. Pointer vers l'hébergeur
   - Netlify : Ajouter un enregistrement DNS
   - Instructions fournies par l'hébergeur

3. Attendre 24-48h (propagation DNS)
